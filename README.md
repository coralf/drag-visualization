## Install

This project uses [node](http://nodejs.org) and [npm](https://npmjs.com). Go check them out if you don't have them locally installed.
```sh
$ node -version
$ npm -version
```

## Usage

### Use bash

``` sh
$ git clone https://github.com/coralf/drag-visualization.git
$ cd drag-visualization
$ npm install
$ npm start
```
